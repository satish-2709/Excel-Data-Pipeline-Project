# 🚀 Excel Data Pipeline Project

## 📌 Overview
This project automates Excel data processing by reading multiple files, merging them, calculating sales percentages, and generating reports with charts.

---

## ✨ Features
- Read multiple Excel files automatically
- Merge and clean data
- Calculate sales percentages
- Generate final Excel report
- Generate visual insights using charts

---

## 🛠 Technologies Used
- Python
- Pandas
- Openpyxl
- Matplotlib

---

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the script:
python main.py
